# Obsidian Storyteller Suite - Agent Instructions

## UI & Icons
- **Icons**: Always use **Lucide icons** via the Obsidian `setIcon` utility to maintain visual consistency with the Obsidian ecosystem.
- **Styling**: Adhere strictly to Obsidian's CSS variables (e.g., `--text-normal`, `--interactive-accent`, `--background-primary`) to ensure compatibility with all themes (Light/Dark).

## Code Standards
- **Patterns**: Maintain established code patterns found in the project (e.g., using `ItemView` for panels, `Modal` for dialogs, and the existing `TimelineRenderer` structure).
- **TypeScript**: Ensure strict typing and follow the project's established module structure.
