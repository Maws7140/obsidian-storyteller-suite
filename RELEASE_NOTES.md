# Release Notes

## Version 1.7.0 - Launch update

### New feature areas
- Campaign and DnD mode with sessions, party state, advanced item effects, faction standing, lore reveals, and branch-driven play
- Character sheet presets with both DnD-themed and note-native layouts

### Major improvements
- Timeline and Gantt redesign with grouped lanes, dependency arrows, progress rendering, and broader interaction fixes
- Campaign boards and map improvements, including SVG support through direct overlay and tiled raster modes
- Compile workflow expansion for custom manuscript pipelines

### Data and setup improvements
- Better entity syncing, group notes, gallery syncing, and wiki-link friendly properties
- First-run onboarding and update-highlights guides inside the plugin

## Version 1.5.6 - Documentation Updates & Calendar Removal

### Documentation and Tutorial Improvements
- Comprehensive updates to tutorial sections covering all available features
- Added detailed tutorials for References, Chapters & Scenes, and Maps management
- Enhanced file structure documentation with complete entity folder listings
- Improved custom folders tutorial with full entity type coverage

### Removed Features
- Removed deprecated Calendar entity type (no longer supported)
- Removed all calendar-related configuration and settings
- Cleaned up calendar references from documentation

### Corrections
- Fixed maps tutorial to accurately reflect plugin capabilities (removed unsupported "draw layers" feature)
